# Criersite (site multipágina)

Site estático (HTML/CSS/JS) pronto para publicar no GitHub Pages.

## Publicar
1. Crie um repositório (ex: `criersite-site`)
2. Faça upload de TODOS os arquivos deste projeto (inclusive `assets/`)
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main | Folder: /(root)
6. Aguarde a publicação e abra o link github.io

## Personalizar
- Textos: edite os arquivos .html
- Cores/estilo: assets/css/styles.css
- Imagens: substitua assets/img/*.svg por suas imagens/capas
